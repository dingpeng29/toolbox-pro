# ToolBox Pro 部署指南

## 方式一：Cloudflare Pages（推荐，免费）

### 前提条件
- GitHub 账号
- Cloudflare 账号

### 步骤

1. **创建 GitHub 仓库**
```bash
cd "D:\baidusyncdisk\ai claude code projects\claude自主创业项目\01-toolbox-pro"
git init
git add .
git commit -m "feat: ToolBox Pro v1.0 - 10 free online tools"
git branch -M main
git remote add origin https://github.com/<你的用户名>/toolbox-pro.git
git push -u origin main
```

2. **连接 Cloudflare Pages**
   - 登录 cloudflare.com
   - 进入 Workers & Pages
   - 创建应用程序 → Pages → 连接到 Git
   - 选择 GitHub 仓库
   - 构建设置：
     - 构建命令：留空（纯静态）
     - 输出目录：`src`
   - 点击"保存并部署"

3. **绑定域名（可选，30元）**
   - 在阿里云/腾讯云购买 .cn 域名
   - 在 Cloudflare Pages → 自定义域 → 添加域名
   - 按指引修改DNS服务器

## 方式二：GitHub Pages（免费备用）

```bash
cd src
git init
git add .
git commit -m "deploy"
git push origin main  # 推送到 gh-pages 分支
```

## 本地测试
直接在浏览器打开 `src/index.html` 即可完整测试所有功能。

## 更新网站
每次修改代码后 push 到 GitHub，Cloudflare Pages 会自动部署更新。
